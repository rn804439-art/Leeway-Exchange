# Leeway Exchange

A React Native mobile application built with Expo for peer-to-peer resource exchange.

## Overview

Leeway Exchange is a mobile platform that enables users to exchange and manage resources efficiently. Built with modern mobile technologies, it provides a seamless user experience across iOS, Android, and Web platforms.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Expo CLI installed globally

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rn804439-art/Leeway-Exchange.git
   cd Leeway-Exchange
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

### Running the App

- **iOS:** Press `i` to open in iOS simulator (macOS only)
- **Android:** Press `a` to open in Android emulator
- **Web:** Press `w` to open in web browser

- `npm start` - Start Expo development server
- `npm run android` - Open in Android emulator
- `npm run ios` - Open in iOS simulator
- `npm run web` - Open in web browser
- `npm run eject` - Eject from Expo (⚠️ irreversible)
